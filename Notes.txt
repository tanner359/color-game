OVERLAP COLOR CALCULATION:
--------------------------
R : 255 + 245 = 500/2 = 250
G : 250 + 50 = 300/2 = 150
B : 170 + 225 = 395/2 = ~200

so

(R1 + R2)/2 = R3


REVERSE:
---------------------------
R : 250 - 245 = 5 * 2 = 10 + 245 = 255
G : 150 - 50 = 100 * 2 = 200 + 50 = 250
B : ~200 - 225 = -25 * 2 = -50 + 225 = ~170 

so

((a - b) * 2) + b = c
